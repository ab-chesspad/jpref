JPref V 0.0.0.1

Целью этого проекта является создание программы, играющей в преферанс, два бота против человека. Игра по-честному, без подглядывания и жульничества.
Предлагаемая версия еще очень далека от конечного результата. Она представляет собой всего лишь графическую платформу для тестирования.
Умеет только торговаться и играть распасы.

Установка
Программа работает на всех OS, поддерживающих стандартную Java, таких, как Windows, macOS, Linux, и т.п.
Для установки программы необходима JRE 1.8 или более новая версия.
Если у Вас JRE не установлена, Вы можете найти все инструкции по установке здесь: www.java.com/en/download/help/download_options.html. Если Вы не планируете участвовать в разработке проекта и просто хотите поиграть с программой, достаточно установить JRE, JDK иметь не обязательно. В этом случае воспользуйтесь уже готовой программой здесь.

Запуск
Откройте Terminal окошко (cmd.exe на Windows) и напечатайте:
java -jar <path to jpref.jar> [список раздач]
Если указан необязательный файл со списком раздач, то программа использует его вместо того, чтобы раздавать карты случайным образом. Это позволяет надежнее тестировать программу.
В качестве списка раздач можно использовать лог, сгенерированный самой программой (см. ниже).
JPref может работать на русском и in English. Если Вы используетев OS in English, но хотите, чтобы JPref работала по-русски, измените запуск программы на:
java -Duser.language=ru -jar <path to jpref.jar> [список раздач]

Использование
Поскольку JPref умеет только играть распасы, то текущая версия в каждом раунде прямо начинает с них, минуя торговлю. Также упрощена процедура хода, который делается однократным нажатием левой кнопки мышки (в настоящей игре будет необходимо подтверждение).
Нажатием правой кнопки мышки вызывается меню:
  • Новая игра - текущая игра обрывается.
  • Переиграть - повторяется текущая раздача.
  • Пуля - текущее положение.
  • Послать лог - см. ниже.
Программа имеет множество настроек, но пока можно только менять расположение и размер окна, все остальные настройки неизменяемые.
Важной особенностью программы является ведение логов, куда записываются все игры. Логи располагаются в logs директории, либо "рядом" с jpref.jar файлом (Mac OS, Linux) либо в <ваша домашняя папка>\jpref (Windows). Новый лог открывается при новом запуске программы или смене дня.
При нажатии на кнопку Комментарии (вторая справа в правом нижнем углу) появляется окошко для комментариев. Особенно важно заносить комментарии об ошибках программы, глупых ходах и т.п.
Весь напечатанный текст попадает в лог, который может быть послан разработчику:
  • Через меню. Ограничение - только текущий лог и только один раз в день.
  • Через github - open new issue.
  • Послать лог файл на ab.jpref@gmail.com

Выход из программы
Рекомендуется закрыть окно программы нажатием на кнопку 'х' в верхнем углу окна. Тогда программа запомнит расположение и размер окна и в следующий раз окно того же размера откроется в том же месте.
